Saved Locations

Adding locations to your saved locations and retrieving them as start or end points.

 
 
 

Saved locations allow you to quickly select from your saved places without having to search each time.

To save a location, first search for a location in either of the start or stop fields, then select the location to set it as a start or stop point.

Once you've have the location set as a start or stop point, tap again into the menu and select Save Location.

In the pop-up message, you can enter a new name for this location, or just use the default name and tap Save.

Your location is now saved. In the future, you can select your new saved location from the menu under Saved Locations.
